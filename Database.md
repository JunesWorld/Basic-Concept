# Database의 기본 개념
-------------------------
## 정의
> * 여러 응용 시스템(프로그램)들의 통합된 정보들을 저장하여 운영할 수 있는 공용(share) 데이터의 집합

## 특성
1. 실시간 접근성
2. 계속적인 변화
3. 동시 공유성
4. 내용 참조

## 데이터 베이스 관리 시스템
> * (Database Management System = DBMS) -> Software
> * 필수 기능
> > 1. 정의 : 데이터 베이스의 논리적, 물리적 구조를 정의
> > 2. 조작 : 데이터를 검색, 삭제, 갱신, 삽입, 삭제
> > 3. 제어 : 데이터베이스의 내용 정확성과 안정성을 유지하도록 제어
